# Finance
 This repository contains scripts for stocks and funds analysis


## ticker_sector.py
Creates a dataframe with all companies symbols (tickers codes) and a new variable with its industry sectors scraping a yahoo finance and wikipedia website.

## time_series_analysis.py
Collets stock historic prices and analyzes evolution over time.
Also plot price variation percentage over the last period.

## funds_analysis.py
Collects daily and monthly price evolution of funds and export it to an excel file

## portfolio.py
Script to create custom portfolios
